# Sustainability Plan: BOSC Community Library

## 1. Resource Maintenance
To ensure the library remains a living resource, we will implement a "Monthly Audit" where student contributors verify that the links and JSON data in the `src/data/` folder still point to active and accurate resources.

## 2. Open-Source Growth
By using the GNU GPL v3 license, we allow other developers to fork this library. This ensures that even if the original team moves on, the code can be adapted for other schools or community centers.

## 3. Low-Cost Infrastructure
The decision to use a "Flat-File" system (JSON) instead of a heavy database means this library can be hosted for free on platforms like GitHub Pages, making it financially sustainable for the long term.
